Portfolio

Pyrenees Basemap

In this project, I downloaded 15 arc minute global DEMs from Viewfinder Panoramas. After mosaicking four tiles, I resampled the final DEM. I then downloaded land cover from the UN Food and Agriculture Organization. ArcMap and Photoshop were used to classify the land data. Illustrator was used to edit the vector data downloaded from Natural Earth.

Fuzzy Soil Map

This project shows soil types (Basco, Elkmound, Council, and Orion) found within Pleasant Valley, Wisconsin based on environmental covariates. To predict soil type, I used slope gradient, elevation, and profile curvature as covariates. SoLIM and ArcMap were used to run the analysis.

Tree Mortality Infographic

In this project, I created a long-form infographic telling the story of tree mortality in California using data from CalFire and the US Department of Agriculture. ArcMap was used to join the data, Indiemapper was used to create the bivariate map, and Illustrator was used to create and style the vector graphics.

Supervisorial Web Map

This project shows supervisor districts within Tuolumne County, CA. The map is interactive by allowing users to hover over different areas to determine what district they are in. First, I created a basemap in Mapbox and uploaded the district geojson files. I then used MapboxGL javascript library to add interactivity to the map.
