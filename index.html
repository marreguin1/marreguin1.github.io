Portfolio

Opioid Story Map Application

In this project, I used mapbox and the leaflet Javascript library to create a web mapping application centered around the United States opioid epidemic. Open Street map and Mapbox are both used as basemaps. I made the application interactive in so far that the states will give users the 2016 opioid deaths per 100,000 people for each state. The states are symbolized to represent higher vs lower death rates. As the user scrolls down the information panel, the map will reposition the extent. The raw data was obtained from the CDC. View the application here.

Fuzzy Soil Map

This project shows soil types (Basco, Elkmound, Council, and Orion) found within Pleasant Valley, Wisconsin based on environmental covariates. To predict soil type, I used slope gradient, elevation, and profile curvature as covariates. SoLIM and ArcMap were used to run the analysis.

Tree Mortality Infographic

In this project, I created a long-form infographic telling the story of tree mortality in California using data from CalFire and the US Department of Agriculture. ArcMap was used to join the data, Indiemapper was used to create the bivariate map, and Illustrator was used to create and style the vector graphics.

Iceland Volcano Eruptions

This project shows volcano eruptions in Iceland since the 1700s. The application is a simple demonstration of mapping temporal data. The eruptions are sorted by date. Graduated points were created based on the Volcanic Eruption Index (VEI). VEI measures the severity of each eruption. Data on eruptions in the 1700s can be sparse and often relies on the written accounts of people who were alive during the volcanic event. Users can use the time slider to cycle through eruption events. The points also have interactive popups that give more information such as number of deaths. Mapbox and leaflet were used to create the application.
